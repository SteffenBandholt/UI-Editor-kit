# Mini-Inspector Demo

Zweck: Sichtbare K1.0-Demo, die nur vorhandene `data-ui-*` Metadaten liest, anzeigt und ein ausgewaehltes Element markiert.

Lokaler Startbefehl:

```powershell
start examples\mini-inspector\index.html
```

Klare Grenze: nur lesen, anzeigen, markieren. Kein Speichern, keine Layoutbearbeitung, keine Fachlogik.
