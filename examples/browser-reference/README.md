# Browser-Referenzanwendung

Start:

```bash
npm run reference:build
npm run reference:browser
```

Die Anwendung ist eine neutrale Referenzanwendung fuer Runtime, Panel, BrowserHostAdapter, ElementRefs, SelectionHost, OverlayHost, BrowserStorage und BrowserBridge. Sie ist keine Produktiv-Ziel-App und enthaelt keine externe Fachanbindung.
