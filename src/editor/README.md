# Editor-Core

Hier wird der wiederverwendbare Editor-Core aufgebaut.

Der Editor-Core bleibt fachneutral.

Er liest nur die im UI-Editor-Vertrag beschriebenen Metadaten.

## Spaetere Bausteine

- Metadaten-Leser
- Auswahl-Engine
- Overlay
- Editor-Panel
- Layout-Werkzeuge
- optionaler UI/PDF-Scope
